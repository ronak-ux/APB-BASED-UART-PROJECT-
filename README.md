APB Based UART Design (Verilog)
Overview
Designed a UART communication system integrated with an APB interface using Verilog HDL.
The project enables serial data transmission and reception with FIFO buffering and module-level verification.
Tools
Verilog HDL
ModelSim
Quartus Prime
Modules
uart_transmitter.v – Serial data transmission
uart_receiver.v – Serial data reception
uart_fifo.v – Data buffering
uart_register.v – Control & configuration registers
uart_top.v – System integration module
Testbenches provided for functional verification.
Features
APB Bus Interface
UART Tx/Rx Communication
FIFO Data Handling
FSM-based Design
Simulation & Verification
